✈️ Ticket Booking Web UI

Ticket Booking Web UI is a Flutter-based web application that demonstrates a modern air ticket booking interface.

The project focuses on creating a clean and responsive UI for a flight ticket and boarding pass, including a barcode representation similar to real airline tickets.

This project is mainly focused on frontend UI design, and the backend booking system is not implemented yet.

🚀 Features

🎫 Air Ticket UI Design – Modern flight ticket layout

📊 Barcode Display – Boarding pass style barcode for ticket verification

🌐 Flutter Web UI – Built using Flutter for web interface

🎨 Clean User Interface – Structured and easy-to-read ticket design

📱 Responsive Layout – Works well across different screen sizes

🛠 Technologies Used

Flutter – Web UI development

Dart – Programming language

Barcode Widget – Used for ticket barcode display

📱 App Concept

The application demonstrates how a flight ticket booking interface could appear in a real system.

The UI displays:

Passenger ticket information

Flight details

Boarding pass design

Barcode for ticket verification

Although the booking system is not functional yet, the UI represents a real-world airline ticket design concept.

⚙️ Installation

Clone the repository:

git clone https://github.com/Jayeshkashyap2201/Ticket-Booking.git

Navigate to the project directory:

cd Ticket-Booking

Install dependencies:

flutter pub get

Run the project:

flutter run
